# Web Expressions ✍️

> Thoughts on Web Development.

A blog by Oliver Ridsdale, built with [Astro](https://astro.build/), [TailwindCSS](https://tailwindcss.com/), Markdown.

## Setup

Install dependencies:

```
$ npm install
```

Run dev server

```
$ npm run dev
```

Build for production, files in /dist

```
$ npm run build
```
